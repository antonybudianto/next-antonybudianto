Hi, here's list of my most used Blender shortcuts:

#### Duplicate object

```
Shift + D, then click
```

#### Add new object

```
Shift + A
```

#### Snap/set 3D Cursor

```
Shift + S
```

#### Move the view

```
Shift + swipe 2 finger
```

#### Move/Rotate/Scale

```
G / R / S, optionally press X/Y/Z for axis lock
```

#### Enter Edit mode

```
Tab, then press 1/2/3 (Vertex/Edge/Face)
```

#### Move to collection

```
M
```

#### Extrude

```
E
```

#### Bevel

```
Cmd + B
```

#### Loop Cut

```
Cmd + R
```

#### Inset

```
I or double II to use individual origin
```

#### Enter Camera view

```
Numpad 0
```

#### Focus to object

```
Numpad .
```
